## Overview

HOI4 Focus Tree Designer is a professional visual drag-and-drop tool for designing and editing Hearts of Iron IV focus trees. This application streamlines the modding workflow by providing an intuitive graphical interface to create complex focus tree structures without manual coding.

### Dedicated to
**Great War - Sultan Sea** - A comprehensive HOI4 total conversion mod

## Usage Guide

### Getting Started

1. **Launch** the application
2. **Create Focus Nodes**
   - Click ➕ button or press `A` to add new focus
   - Double-click to edit properties
   - Drag to move nodes on canvas

3. **Create Connections**
   - `C` - Add prerequisite link (yellow)
   - `E` - Add mutual exclusive link (red)
   - Right-click to cancel connection mode

4. **Configure Properties**
   - Edit in right panel while node is selected
   - Set name, cost, conditions, rewards, AI behavior
   - Click "APPLY CHANGES" to save

5. **Export**
   - Press `Ctrl+E` or File → Export HOI4 .txt
   - Choose filename and location
   - Import to your HOI4 mod

### Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+N` | New Project |
| `Ctrl+O` | Open Project |
| `Ctrl+S` | Save Project |
| `Ctrl+E` | Export to HOI4 |
| `A` | Add Focus Node |
| `C` | Prerequisite Connection |
| `E` | Mutual Exclusive |
| `Delete` | Delete Selected |
| `F` | Fit All in View |
| `Middle-Click` + Drag | Pan Canvas |
| `Scroll` | Zoom In/Out |

### Canvas Navigation
- **Pan:** Middle-click and drag
- **Zoom:** Mouse wheel scroll
- **Select:** Click nodes or drag selection box
- **Multi-select:** Hold `Shift` while clicking

## Project File Format

Projects are saved as `.hftp` files (JSON-based) containing:
- Focus node configurations
- Node positions and layout
- Connection relationships
- Tree metadata

These can be version-controlled and shared with collaborators.

## Troubleshooting

**Issue:** Application won't start
- **Solution:** Ensure Windows 10+ and 64-bit system. Re-run installer.

**Issue:** Cannot find save location
- **Solution:** Check file permissions in `Documents` folder. Run as Administrator if needed.

**Issue:** Export creates invalid HOI4 syntax
- **Solution:** Verify focus IDs use valid characters. Export locations must be integers. Check HOI4 documentation.
